Interview Checklist

  • 1. Write a Query to Find Duplicates
    Answer: Use GROUP BY and HAVING COUNT > 1 to find duplicate entries in a table.
  • 2. Write a Query to Fetch the Second Highest Salary
    Answer: Use ORDER BY DESC and LIMIT 1 OFFSET 1 to get the second-highest salary.
  • 3. Explain ACID Properties
    Answer: ACID stands for Atomicity, Consistency, Isolation, and Durability, ensuring reliable transactions.
  • 4. What is a Primary Key and Foreign Key?
    Answer: A Primary Key uniquely identifies each record; a Foreign Key links two tables together.
  • 5. Write a Query to Find the Number of Employees in Each Department
    Answer: Use GROUP BY department_id and COUNT(*) to get the employee count per department.
  • 6. Explain the Difference Between WHERE and HAVING
    Answer: WHERE filters rows before grouping; HAVING filters groups after grouping.
  • 7. Write a Query to Join Two Tables
    Answer: Use INNER JOIN, LEFT JOIN, or RIGHT JOIN to retrieve data from multiple tables.
  • 8. Write a Query to Get Employees Who Joined Last Month
    Answer: Use DATE_SUB() and BETWEEN to filter for employees who joined last month.
  • 9. Explain Normalization and Denormalization
    Answer: Normalization reduces redundancy by splitting data; denormalization combines tables for faster reads.
  • 10. Write a Query to Remove Duplicates from a Table
    Answer: Use ROW_NUMBER() or DELETE with JOIN to remove duplicate rows from a table.
  • 11. What is the Difference Between DELETE and TRUNCATE?
    Answer: DELETE removes rows with a WHERE clause; TRUNCATE deletes all rows quickly but cannot be used with a WHERE clause.
  • 12. Write a Query to Find the Highest and Lowest Salary
    Answer: Use MAX(salary) and MIN(salary) to retrieve the highest and lowest salary.
  • 13. Explain the Concept of an Index
    Answer: Indexes improve the speed of data retrieval but can slow down inserts, updates, and deletes.
  • 14. Write a Query to Display Only Odd or Even Records
    Answer: Use ROW_NUMBER() with modulo (%) to filter odd/even records.
  • 15. What is the Difference Between UNION and UNION ALL?
    Answer: UNION removes duplicates; UNION ALL includes all duplicates.
  • 16. Write a Query to Find Employees with Salaries Above Average
    Answer: Use a subquery to calculate the average salary and filter employees with salaries above this value.
  • 17. Explain the Difference Between INNER JOIN and OUTER JOIN
    Answer: INNER JOIN returns only matching records; OUTER JOIN returns all records with NULL for non-matches.
  • 18. Write a Query to List All Table Names in a Database
    Answer: Use SELECT table_name FROM information_schema.tables to list table names.
  • 19. What is a View, and How is it Used?
    Answer: A view is a virtual table based on a query; it simplifies data access without storing data itself.
  • 20. Write a Query to Add a Column to an Existing Table
    Answer: Use ALTER TABLE table_name ADD column_name data_type to add a column.
  • 21. What is the Difference Between a Primary Key and a Unique Key?
    Answer: A primary key uniquely identifies each row and does not allow NULLs; a unique key also enforces uniqueness but can allow NULLs.
  • 22. Write a Query to Convert Rows to Columns
    Answer: Use CASE statements or the PIVOT function to convert rows to columns.
  • 23. Explain the Concept of a Foreign Key Constraint
    Answer: A foreign key constraint enforces a link between two tables, maintaining referential integrity.
  • 24. Write a Query to Remove a Column from a Table
    Answer: Use ALTER TABLE table_name DROP COLUMN column_name to remove a column.
  • 25. What is the Use of the COALESCE Function?
    Answer: COALESCE returns the first non-NULL value in a list of arguments.
  • 26. Write a Query to Calculate the Total Salary of Employees in Each Department
    Answer: Use SUM(salary) with GROUP BY department_id to calculate total salary by department.
  • 27. Explain the Purpose of the EXISTS Keyword
    Answer: EXISTS checks if a subquery returns any rows and is used for efficient conditional checks.
  • 28. Write a Query to Find the Length of a String
    Answer: Use LENGTH() function to get the number of characters in a string.
  • 29. Explain the Difference Between CHAR and VARCHAR
    Answer: CHAR is a fixed-length string, while VARCHAR is a variable-length string up to a specified limit.
  • 30. Write a Query to Fetch the Top N Records
    Answer: Use LIMIT n to get the top N records in MySQL or TOP N in SQL Server.
